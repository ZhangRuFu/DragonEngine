# DragonEngine
## 3DGameEngine - OpenGL
使用OpenGL作为图形接口（将来也可能支持DirectX）的3D游戏引擎

---

### 目前已实现：
- 漫游
- 模型加载
- 骨骼动画播放

---

### 正在开发：
- 动画状态机
- 基本UI

---

### 未来：
类Unity3D组件式游戏引擎，具有渲染组件、AI组件和物理组件
